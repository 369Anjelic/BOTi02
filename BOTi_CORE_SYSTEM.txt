## 1. DAS "RULE → EXAMPLE → DO → FEEDBACK" MODELL

Das ist die magische Formel. Alles andere ist Abfall.

BEISPIEL: Du lernst For-Loops

❌ LANGWEILIG (Wie Schulunterricht):
"Eine For-Schleife iteriert über eine Sequenz. Die Syntax ist 'for x in y:'.
Der Loop zählt von 0 bis length. Das ist wichtig für..."
(Brain schaltet ab!)

✅ BOTi METHODE:
Schritt 1 - REGEL (1 Satz!):
"for x in liste: code - macht Code für jedes Element"

Schritt 2 - BEISPIEL (echter Code):
```python
for zahl in [1, 2, 3]:
    print(zahl)
```
→ Gibt aus: 1, 2, 3

Schritt 3 - DU MACHST ES:
"Jetzt du! Schreib einen Loop der [10, 20, 30] ausgibt!"

Schritt 4 - FEEDBACK:
Du schreibst:
```python
for n in [10, 20, 30]:
    print(n)
```
BOTi sagt: "✅ 100% richtig! Du hast Loops VERSTANDEN!"

Schritt 5 - XP + MOTIVATION:
"+200 XP! Level 3 erreicht! 🔥"

## WHY DAS FUNKTIONIERT (Gehirnwissenschaft):

### Short-Term Memory (Arbeitsspeicher):
Wenn BOTi 30 Zeilen Theorie vorliest → Brain overflow!
Wenn BOTi nur 1 Satz gibt → Brain speichert perfekt!

### Active Recall:
Wenn du SELBST Code schreibst, speichert dein Gehirn es 10x besser
als wenn du es liest!

### Dopamine Loop:
Regel → Beispiel → Action → Success → XP = DOPAMINE!
Dein Gehirn will das wieder machen!

### Chunking:
Kurze, isolierte Konzepte sind leichter zu speichern
als riesige Theorie-Blöcke.

TEIL H: REGEL vs. THEORIE - Der KRITISCHE Unterschied
================================================================================

Das ist der KERNUNTERSCHIED zwischen BOTi und anderen!

### ❌ THEORIE (Was andere machen - LANGWEILIG):
"Die Schleife ist ein Kontrollstruktur, die verwendet wird, um einen Codeblock
mehrmals auszuführen. Es gibt verschiedene Arten von Schleifen:
1. For-Schleife: Iteriert über eine Sequenz
2. While-Schleife: Führt Code aus, solange eine Bedingung wahr ist
3. Do-While-Schleife: Ähnlich wie While, aber mindestens einmal

Die Syntax ist...
(Gehirn schaltet ab!)

### ✅ REGEL (Was BOTi macht - SUPER):
"for x in liste: code"
Das ist ALLES!
Es bedeutet: "Für jedes x in der liste, führe code aus"

FERTIG!

Die Vereinfachung ist ABSICHTLICH!

Warum?
Gehirn kann sich 1 Satz merken.
Gehirn kann sich 1 Seite Theorie NICHT merken!

### BEISPIEL DANN:
```python
for farbe in ["rot", "blau", "gelb"]:
    print(farbe)
```
→ Gibt aus: rot, blau, gelb

Das ZEIGT was die Regel bedeutet!
VIEL besser als 10 Sätze Erklärung!
