SCHRITT 1: CLAUDE API KEY BESORGEN (KOSTENLOS!)
──────────────────────────────────────────────────

1. Gehe zu: https://console.anthropic.com
2. Klick oben rechts: "Get API Key"
3. Sign up mit deiner Email (kostenlos!)
4. Bestätige Email
5. Klick "Create Key"
6. Du siehst: sk_ant_xxxxxx...
7. COPY! (Ctrl+C / Cmd+C)


SCHRITT 2: ENVIRONMENT-DATEI ERSTELLEN
────────────────────────────────────────

1. Öffne Texteditor (Notepad, VS Code, etc.)
2. Schreib rein:

   ANTHROPIC_API_KEY=sk_ant_xxxxx

   (Ersetze xxxxx mit deinem echten Key!)

3. Speichern als: .env
4. In den gleichen Ordner wie BOTi_MEGA_FINAL.html


SCHRITT 3: BOTi MIT API STARTEN
────────────────────────────────

Option A: Mit Python (einfachste!)
──────────────────────────────────

1. Command Prompt / Terminal öffnen
2. Zum Ordner navigieren wo BOTi ist:

   cd Desktop
   (oder wo deine BOTi-Datei ist)

3. Python Server starten:

   python -m http.server 8000

4. Browser öffnen: http://localhost:8000/BOTi_MEGA_FINAL.html
5. BOTi lädt mit API! 🚀


Option B: Mit Node.js (schneller!)
──────────────────────────────────

1. Node.js installieren (von nodejs.org)
2. Command Prompt / Terminal:

   npm install -g http-server
   http-server

3. Browser: http://localhost:8080/BOTi_MEGA_FINAL.html
4. BOTi mit AI lädt! 🤖

================================================================================
                    🆘 HÄUFIGE PROBLEME
================================================================================

F: "Die Datei öffnet sich nicht!"
A: 1. Rechtsklick auf BOTi_MEGA_FINAL.html
   2. "Öffnen mit" → Browser (Chrome, Safari, Firefox, Edge)
   3. Sollte jetzt funktionieren!

F: "Ich sehe nur Code statt die App!"
A: Das ist normal wenn du es als Text öffnest.
   Lösung: Rechtsklick → "Mit Browser öffnen"

F: "Code Editor ist leer!"
A: Scrolle runter! Der Editor ist unter dem Example Box
   oder das Fenster ist zu klein.
   Lösung: Browser-Zoom vergrößern (Ctrl+Plus)

F: "Ich habe keine Internet, kann ich noch spielen?"
A: JA! BOTi funktioniert 100% offline!
   Die HTML-Datei enthält alles!

F: "Wie speichert BOTi meinen Fortschritt?"
A: Momentan wird der Fortschritt nur in der aktuellen
   Session gespeichert. Nach Reload ist alles weg.
   Nächste Version: Fortschritt speichern! 📅

F: "Kann ich BOTi auf mehreren Geräten spielen?"
A: Ja! Einfach die HTML-Datei auf alle Geräte kopieren.
   oder: GitHub pushen (wenn ich das mache) →
   Überall spielbar via URL! 🌍
