SCHRITT 1: React Native App Struktur
──────────────────────────────────────

Code-Struktur wäre:

```javascript
// App.js
import React, { useState } from 'react';
import { View, Text, ScrollView, TouchableOpacity } from 'react-native';
import { SafeAreaView } from 'react-native-safe-area-context';

export default function App() {
  const [currentLang, setCurrentLang] = useState('python');
  const [level, setLevel] = useState(1);
  const [xp, setXp] = useState(0);

  return (
    <SafeAreaView style={{ flex: 1 }}>
      <View style={{ padding: 20 }}>
        <Text style={{ fontSize: 24, fontWeight: 'bold' }}>
          🤖 BOTi Mega
        </Text>
        
        {/* Language Selector */}
        <ScrollView horizontal>
          {/* 50+ Language Buttons */}
        </ScrollView>
        
        {/* Challenge Area */}
        <View style={{ marginTop: 20 }}>
          {/* Challenge Content */}
        </View>
        
        {/* Stats */}
        <View style={{ marginTop: 20 }}>
          <Text>Level: {level}</Text>
          <Text>XP: {xp}</Text>
        </View>
      </View>
    </SafeAreaView>
  );
}
```


SCHRITT 2: App bauen & testen
──────────────────────────────

Mit Expo (einfachste Methode):

```bash

# Neues Projekt

cd BOTi-Mobile

# QR-Code mit iPhone-Kamera scannen

```

→ App läuft sofort auf iPhone in Development Mode!
