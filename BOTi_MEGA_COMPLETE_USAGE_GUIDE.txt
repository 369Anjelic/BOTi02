                  SCHRITT 5: CODE SCHREIBEN & SUBMITTEN
================================================================================

TIPPS ZUM EFFIZIENTEN CODING:
──────────────────────────────

1. LESE DIE RULE
   Nicht das Code Editor anschauen!
   Rule sagt dir was du tun musst!

2. SCHAU DAS EXAMPLE
   Sieh wie es funktioniert
   Nutze als Template!

3. SCHREIB DEINE VERSION
   Löse die TASK nicht das Example!

4. SUBMIT & PRÜFE
   Wenn richtig → XP!
   Wenn falsch → Hint anschauen!


BEISPIEL - Python Variables Challenge:

RULE sagt:
  "name = value - stores data"

EXAMPLE zeigt:
  name = "Alice"
  print(name)

TASK verlangt:
  "Create variable 'dein_name' and print it"

DEINE LÖSUNG:
  dein_name = "Max"
  print(dein_name)

SUBMIT →
  ✅ CORRECT! +100 XP!


HÄUFIGE FEHLER:
────────────────

❌ FALSCH: Code genau 1:1 vom Example kopieren
   → Challenge verlangt DEINE Version!

❌ FALSCH: Variable "Alice" statt "dein_name"
   → Lies die Task genau!

❌ FALSCH: print() vergessen
   → Sieh die Rule & Example!

✅ RICHTIG: Logic verstehen → eigener Code → Submit

BONUS: CLAUDE API CHALLENGES

Wenn du die App erweitern möchtest:

```javascript
async function getAIFeedback(prompt) {
  const response = await fetch('https://api.anthropic.com/v1/messages', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'x-api-key': 'YOUR_API_KEY'
    },
    body: JSON.stringify({
      model: 'claude-opus-4-20250514',

  });
  return response.json();
}
```
